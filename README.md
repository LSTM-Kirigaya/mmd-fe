# mmd-fe (multimodal medical database front-end)

## 项目运行

安装：

```bash
npm i
```

运行：

```bash
npm run serve
```

## 项目结构

```
src
├─📁api         后端 API
├─📁assets      静态资源
│  ├─📁css      层叠样式表
│  ├─📁fonts    字体
│  └─📁image    图像
├─📁components  自定义全局通用组件
├─📁hook        自定义函数库
├─📁i18n        多语言配置
├─📁mock        mock数据配置
├─📁router      路由
├─📁store       状态库
├─📁types       全局自定义 type
└─📁views       路由组件
```
